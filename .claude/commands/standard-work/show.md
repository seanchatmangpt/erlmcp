# Command: /standard-work-show

## Purpose
**Japanese**: 標準作業 (Hyōjun Sagyō - Standard Work)
Show standard work documentation for any stage.

## Usage
```bash
/standard-work-show <stage>
```

## Examples
```bash
/standard-work-show build
```

Shows Stage 6 standard work from `docs/tcps/STANDARD_WORK.md`.
