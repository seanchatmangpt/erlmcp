# Claude Code CLI - Diátaxis Documentation

This directory contains comprehensive documentation for Claude Code CLI, organized according to the Diátaxis framework. The documentation provides four complementary approaches to understanding and implementing Claude Code CLI in your development workflow.

## What is Diátaxis?

The Diátaxis framework categorizes technical documentation into four quadrants based on user needs:

1. **EXPLAIN** - Conceptual understanding and theoretical foundation
2. **HOWTO** - Practical implementation and step-by-step guidance
3. **REFERENCE** - Technical specifications and authoritative information
4. **TUTORIAL** - Learning through example and guided practice

This ensures documentation addresses different learning styles and use cases effectively.
